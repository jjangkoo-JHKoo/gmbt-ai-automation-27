## 📋 프로젝트 소개

마케팅, 영업, 기획팀의 보고서 작성 시 이미지 선택에 소요되는 
**시간과 비용을 AI 자동화로 혁신**하는 프로젝트입니다.

### 핵심 효과
- ⏱️ **시간 절감**: 40분 → 30초 (94% 단축)
- 💰 **비용 절감**: 월 60만원 → 월 3만원 (95% 절감)
- 🚀 **생산성 향상**: 월 10건 → 월 37건 (3배)
